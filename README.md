# Ice Password Generator

A secure, user-friendly password generator built with HTML, CSS, and JavaScript. Create strong, unique passwords with customizable options and real-time strength assessment.

## Features

- **Password & PIN Generation**: Choose between generating passwords or PINs
- **Customizable Options**:
  - Adjustable length (4-64 characters)
  - Include/exclude uppercase letters, lowercase letters, numbers, and symbols
  - Avoid ambiguous characters option
  - Add custom words for memorable passwords
- **Real-time Strength Meter**: Visual feedback on password strength
- **Copy to Clipboard**: Easily copy generated passwords
- **Download as Text File**: Save passwords securely
- **Dark/Light Mode Toggle**: Comfortable viewing in any environment
- **Security Tips**: Built-in best practices for password management
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Select Password or PIN mode
2. Adjust the length using the slider
3. Choose which character types to include
4. Optionally add custom words
5. Click "Generate Password" to create a secure password
6. Copy or download the result

## Security Features

- Uses cryptographically secure random number generation
- Avoids commonly confused characters (e.g., 0/O, 1/l/I)
- Provides strength assessment based on length and character variety
- Includes educational security tips

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (ES6+)
